AnyEvent-Yubico

  • Releases

AnyEvent based Perl extension for validating YubiKey OTPs. Though AnyEvent is used internally, the module does not impose any particular coding style on the caller. Provides both blocking and non-blocking versions of OTP verification.

INSTALLATION

To install this module type the following:

perl Makefile.PL make make test make install

DEPENDENCIES

This module requires these other modules and libraries:

AnyEvent::HTTP, UUID::Tiny, MIME::Base64, Digest::HMAC_SHA1, URI::Escape, Net::SSLeay
